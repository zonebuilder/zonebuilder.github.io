JUL News Reader - Vuetify


For a live demo, click HERE


Each project/module of this app builds the template for a NuxtJS SFC in a Node ES6 project. The Node project is located in Designer's 'assets/examples/nesreader.vuetify/' working folder.
The module -> file mapping is as follows:
  • Articles -> pages/index.vue
  • Entry -> pages/entry.vue
  • Layout -> layouts/defaut.vue
If you run the hot reloading development server ( npm run dev ), each save of a project in Designer will update the Node application. You may inspect the convert.js file to see how the Designer generated template is converted to the SFC inline.
Note the Vue routing mode setup in nuxt.conf.js fine.
For the actual development, you need to have Node.js installed, open a console window in the Node project folder, and type the following shell commands: 
npm install
npm run dev
The development app will start at http://localhost:3000/ . Use an up-to-date browser to run the development app or the live demo.
To build a production app, use the command: 
npm run generate
as specified in the Vuetify Docs, and in the Nuxt.js Docs. Good luck and have a nice development :-)
