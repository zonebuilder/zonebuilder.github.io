JWL Components Demo

These are two ways of using JWL components: with custom HTML elements (web components W3C standard), or with element wrappers. The element wrappers expose the standard DOM properties and methods of the element, but can augment them the same way as the custom elements do.

The required code for both use cases is pretty similar, as you can see below.
